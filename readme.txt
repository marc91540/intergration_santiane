Partie integration :
	Vous devez reproduire le template fourni. Si les tailles n'ont pas besoin d'être identiques, les différents ratios sont à respecter.
	Le rendu devra être responsive et vous êtes libre de choisir la meilleure façon de faire évoluer votre page selon le support client.
	Vous êtes libre d'utiliser ou non une librarie.
Partie dynamique :
	Le centre de notre page devra être completé grâce au retour d'une API. Cette API nous retourne une liste d'agents ainsi que leurs ligues.
	spider.in.santiane.fr/api/v1/leagues/agents?fields={}

Rendu :
	Une fois terminé, vous devez envoyer votre code ainsi que toutes les dépendances necessaires sur une branche nommée {votreNom_votrePrenom}
